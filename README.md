# Hey, I'm Marc 👋🏻 Welcome aboard.

📍 **Spaceship Earth**

**Short version**: I make things with code, machines, and paper. Sometimes all three at once.

**Longer version**: I got my start in the late 90s [demoscene](https://en.wikipedia.org/wiki/Demoscene), as part of a crew called [Mandarine](https://www.pouet.net/groups.php?which=5). If you don't know what the demoscene is, imagine a bunch of teenagers competing to make computers do beautiful, impossible things in as few kilobytes as possible. It broke my brain in the best way. I never really recovered.

Since then I've spent about two decades working in tech, writing code, building products, and eventually leading engineering and product teams. I've lived in Amsterdam, Paris, Montreal, Berlin. Never quite landing anywhere, always picking up new frequencies along the way

In 2014 I co-founded [POLAAR](https://polaarsounds.com), an electronic music label for the sounds that live at the edges. Techno, bass music, experimental stuff. The things that happen between genres.

More recently I've fallen deep into generative art and pen plotting. I write code that tells a robotic arm where to drag ink across paper. I also draw by hand, the old fashioned way.

So yeah. I work in tech, I make art, and I live on Spaceship Earth.
